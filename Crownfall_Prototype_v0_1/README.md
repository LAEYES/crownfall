
# Crownfall – Prototype v0.5

Minimal playable prototype (simulation-first).

## How to run
```bash
python main.py
```

## What this prototype does
- Turn-based kingdom simulation with seasonal effects
- Text-based UI with status bars, focus policy, and council menu choices
- Player actions (build, feast, tax, recruit, parade, aid, policy focus)
- Resources, population, morale, loyalty, unrest, stability, and focus tracking
- Dynamic events
- Chronicle generation

No graphics yet – this is the mechanical core.
